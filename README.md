# Generador de Informes de Revision Mensual

Aplicacion web estatica para generar informes comparativos mensuales de nomina desde archivos Excel.

## Uso

1. Abrir la pagina publicada en GitHub Pages.
2. Cargar el Excel del mes anterior.
3. Cargar el Excel del mes actual.
4. Descargar el informe requerido.

Los archivos se procesan localmente en el navegador.
